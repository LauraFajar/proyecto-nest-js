# Proyecto AGROTIC

Este repositorio contiene el código fuente completo del proyecto AGROTIC, una solución integral para la gestión agrícola.

Para obtener información detallada sobre cada componente del proyecto, por favor, consulta los archivos `README.md` específicos ubicados en las siguientes carpetas:

*   `backend`: Contiene la lógica del servidor y la API.
*   `frontend-web`: Contiene la aplicación web para la interfaz de usuario.
*   `config-iot`: Incluye la configuración y el código para los dispositivos IoT.
*   `documentacion`: Almacena la documentación de usuarios y documentación técnica del proyecto.
*   `frontend-movil`: Contiene el código del frontend de la aplicación móvil.
